Brief History Of Html

  1. Tim Berners-Lee invents the World Wide Web and create the first version of HTML at CERN in 1989.
  2. HTML 1.0 is released, allowing for basic text formatting and hyperlinks in 1991.
  3. HTML 2.0 is released, adding features like forms,tables,and image support in 1993
  4. HTML 3.0 i released, introducing tables,applets,and improved formatting.
  5. HTML 5.3 is released, with ongoing improvement and additions
